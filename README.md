# CleanRecycle
This script will purge the Agile Central recycle bin of artifacts

### Installation:
`pip install requests`

`pip install pyral`

### Usage:
`python clean_recycle.py --server=<hostname> --user=<login ID> --password=<password> --workspace="<workspace>" --project="<project>"`
